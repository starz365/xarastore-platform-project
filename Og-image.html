🖼️ Open Graph Image Preview (1200x630)

📋 Save Instructions

Method 1: Screenshot (Easiest)

  1. Take a screenshot of the image above
  2. Crop to exactly 1200x630 pixels
  3. Save as og-image.png and twitter-image.png
  4. Place in public/ directory

Method 2: Convert SVG to PNG

  1. Copy the SVG code below
  2. Visit svgtopng.com or cloudconvert.com
  3. Set dimensions to 1200x630
  4. Convert and download
  5. Save as both:
    • public/og-image.png
    • public/twitter-image.png

Method 3: Use Browser DevTools

  1. Right-click the image above → Inspect
  2. Right-click the <svg> element in DevTools
  3. Copy → Copy element
  4. Paste into an online SVG to PNG converter
💡 Pro Tip: The same image can be used for both og-image.png and twitter-image.png - just save it twice with different names!
