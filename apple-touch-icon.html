📱 Apple Touch Icon Instructions

Method 1: Save as PNG (Recommended)

  1. Right-click on the icon above
  2. Select "Save Image As..." or take a screenshot
  3. Save as apple-touch-icon.png
  4. Move to public/icons/apple-touch-icon.png

Method 2: Use online converter

  1. Right-click the SVG code below and copy
  2. Visit svgtopng.com
  3. Paste the SVG code and convert to 180x180 PNG
  4. Download and save to public/icons/apple-touch-icon.png

Method 3: Screenshot

  1. Take a screenshot of the icon above
  2. Crop to 180x180 pixels
  3. Save to public/icons/apple-touch-icon.png
